# 🧮 Advanced Calculator

A modern and responsive calculator built using **HTML, CSS, and JavaScript**. This project supports basic arithmetic operations along with additional features such as calculation history, keyboard support, dark/light mode, and scientific functions.

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📊 Percentage Calculation
* √ Square Root
* x² Power Function
* 📜 Calculation History
* ⌨️ Keyboard Support
* 🌙 Dark/Light Mode
* 📱 Fully Responsive Design
* 🎨 Modern Glassmorphism UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Calculator/
│
├── index.html
├── style.css
└── script.js
```

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Open the project folder.

3. Run `index.html` in your browser.

## ⌨️ Keyboard Shortcuts

| Key       | Function              |
| --------- | --------------------- |
| 0-9       | Number Input          |
| + - * /   | Operations            |
| Enter     | Calculate             |
| Backspace | Delete Last Character |
| Escape    | Clear Display         |
